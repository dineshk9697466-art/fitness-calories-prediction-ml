# Fitness Calories Prediction (ML Project)

## 📊 Problem Statement
Predict the number of calories burned based on user activity and health-related features such as age, duration, heart rate, and body parameters.

## ⚙️ Approach
- Data Cleaning & Preprocessing
- Feature Scaling (Normalization / Standardization)
- Exploratory Data Analysis (EDA)
- Model Building using Multiple Linear Regression
- Model Evaluation using regression metrics (R², MAE, RMSE)

## 📈 Results
- R² Score: 0.76
- Mean Absolute Error (MAE): 31.66
- Root Mean Squared Error (RMSE): 42.79

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn

## 📌 Key Insights
- Duration and heart rate are major factors influencing calorie burn
- Feature scaling improved model performance
- Model achieved reliable prediction accuracy with low error rates

## 📌 Conclusion
The Multiple Linear Regression model performed effectively in predicting calories burned, demonstrating strong relationships between physical activity features and output.
